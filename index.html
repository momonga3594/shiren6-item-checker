識別手段の基本方針
  • 漢識別（使用識別）：とりあえず使って結果から推測。武器・盾・巻物・草・お香・押す壺はそのまま識別状態になる。デメリット効果のリスクがあるため、フロアの用事を済ませて階段の上で行うのが理想。
  • 値段識別：店やナゾーさんで値段を調べて絞り込む。ノーリスクで手持ち全てを調べられる。買値を名前につけておくと便利（売値×2.5＝買値）。
  • 識別の巻物：万能の確定識別。ラッキーで壺の外の持ち物が全識別。祝福された識別の巻物は確定でラッキー発生。腕輪・壺への使用優先度が高い。
  • 識別の壺：複数識別できるが、取り出すまで使用不可・壺とお香は識別不可。
  • 特製おにぎり：効果の1つ「拾い識別状態」を引けば床置き→拾い直しで全識別。HP満タン・腹減らず・しのび足状態で食べると発生率UP。
  • 鑑定師の腕輪：拾うだけで識別される最強手段（超レア）。
  • マゼルン・合成識別：未識別装備同士を合成すると合成後は識別状態に。材料に識別済みが1つでもあれば呪いの有無も判明。
  • 店の商品で漢識別：杖・壺は（基本価格＋回数×増量価格）なので、使った回数分だけ払えば安く済む。場所がえ・感電・水鉄砲は店で事故のもと。

値段識別の基礎知識

  • 呪われたアイテムは買値・売値が13%引き（端数切下げ）→ 端数の値段は呪いのサイン（例: 1305/522、3045/1218の腕輪は垂れ流し系）。
  • 祝福されたアイテム（草・巻物）は値段2倍。100Gの草が祝福で200Gに見えることがあるため、店内での軽率な漢識別は泥棒事故のもと。
  • 店内では持ち物の説明文右上に売値が表示される。
  • 食料は最初から全て識別済み。雑草・脱出/白紙/ぬれた巻物も識別済み。
腕輪の識別チャート

垂れ流し・金垂れ流し・胃縮小の腕輪は確定で呪われている。1500Gの店売り腕輪は高飛び・爆発の可能性があるため試着厳禁。

装備した瞬間に分かる

  • ちからが増えた → ちからの腕輪
  • 最大満腹度が増えた/減った → 胃拡張/胃縮小の腕輪
  • 敵の位置が分かる → 気配察知 ／ 道具の位置 → 道具感知 ／ 両方 → 透視の腕輪

ターン経過で分かる

  • 毎ターン経験値+1 → しあわせの腕輪
  • 回復と腹減りが早い → 回復の腕輪
  • ワープした → 高飛び ／ 爆発した → 爆発の腕輪
  • アイテム/ギタンを落とした → 垂れ流し/金垂れ流しの腕輪

投げる・撃つと分かる

  • 壁を抜けた → 遠投 ／ 反射した → ボヨヨン ／ 2連射 → 連射
  • 必ず外れる → ヘタ投げ ／ 必ず当たる → 百発百中

特定の行動で分かる

  • 壁に入れた → 壁抜け ／ 水上を歩けた → 水グモ ／ 水も穴も罠無効 → 浮遊
  • 会心も痛恨も出る → 諸刃 ／ 痛恨だけ受ける → 痛恨の腕輪
  • 敵と位置が入れ替わる → すれちがい ／ 仮眠敵が起きない → 忍び足
  • 拾ったら識別された → 鑑定師の腕輪

識別が難しいもの

  • 混乱/睡眠/呪い/毒を打ち消した → 各よけ系腕輪
  • 錆びず印消しも受けない → 錆よけ ／ 弾かれない → 弾きよけ
  • 大砲の威力UP → 大砲強化 ／ 隣接だけで隠し壁が壊れた → 裏道の腕輪
杖の識別チャート（振って確認）

モンスターに振ればほぼ判別可能。2マス以上離れて振る（幸せ・感電・土塊・飛びつき対策）／複数の敵がいる時・バッター種相手は避ける。

ダメージ量で分かる

  • 25ダメージ → 感電の杖
  • 10ダメージ → トンネルの杖
  • 5ダメージ → 吹き飛ばしの杖（壁際の敵）
  • 2ダメージ → 土塊の杖（隣接した敵）

状態異常で分かる

  • 鈍足になった → 鈍足の杖 or ガイコツまどうの杖（複数回で確認）
  • かなしばり → かなしばりの杖 ／ 封印 → 封印の杖 ／ 身代わり → 身代わりの杖
  • ワープ後かなしばり → 一時しのぎの杖 ／ 加速した → 加速の杖

移動・変化で分かる

  • 敵が吹き飛んだ → 吹き飛ばし ／ 位置が入れ替わった → 場所がえ
  • 着弾点まで自分が移動 → 飛びつき ／ 階段方向に弾が飛んだ → 導き
  • レベルUP → 幸せ ／ レベルDOWN（またはメッセージ） → 不幸 ／ 桃まん化 → 桃まんの杖

すぐ分からないもの

  • 被ダメージ時に相手も同ダメージ → 痛み分けの杖
  • 転倒時に発動 → 転ばぬ先の杖
  • 壁に振って「重くて動かない」→ 飛びつき ／ 何も出ない → 場所がえ（ドスコイ状態時）
  • どれでもない → ただの杖

値段でも絞れる

  • 買値1400G以上 → 痛み分け・一時しのぎ・幸せ・不幸・ガイコツまどう
  • 買値2400G以上 → 身代わり・桃まん
巻物の識別のコツ
  • 読めば識別されるので漢識別が基本。マイナス枠：くちなし・拾えず・罠・敵加速・迷子・生物集合・バクチ・魔物部屋。フロア巡回を終えて階段の上で読むこと。
  • 「どれに使う？」と聞かれたら → 呪い・銀封印・印消し・おにぎり・識別・おはらい・天/地の恵み・メッキ・銀はがし・印/壺増大・吸い出しのどれか。おにぎり対策に不要装備を選ぶのが安全。
  • 目安：銀封印300G／印消し・おにぎり400G／呪い1000G。
  • 聖域の巻物は置くと広がるので読む前に置いて確認（置くだけでは未識別のまま）。
  • 白紙の巻物は「書く」を選ぶだけで呪い・祝福が判明。
  • 全滅(3000G)とねだやし(10000G)は値段で確定。
  • 店売りの魔物部屋の巻物を店内で読むとワープ→強制泥棒になる。1000Gの巻物は買ってから読むこと。
草・種の識別（飲む／投げる）

飲めば識別される（投げ当てでは識別されない）。回復草は最大HP上昇があるためHP満タンで飲むとお得。

敵に投げた時の反応で推測（メモ→後から名付け）

  • HP25回復 → 薬草 ／ 100回復 → 弟切草 ／ 200回復+状態回復 → いやし草 ／ 全回復 → 命の草
  • 攻撃力が上がった → ちからの草（短い効果音）or パワーアップ草（長い効果音）
  • 草エフェクト後に2ダメージ → 胃拡張/胃縮小の種・毒消し草
  • バシッと当たって2ダメージ → 復活の草 or 呪われた草（要注意）
  • 毒サソリ種に50ダメージ → 毒消し草 ／ おばけ大根種の攻撃UP → 毒草
  • ゴースト系には回復草がダメージになる（命の草は残りHP分）

レベリング活用

混乱草・暴走の種・目つぶし草は「あなぐらマムル」等に投げてレベルアップさせ、経験値稼ぎに使える（序盤限定の小技）。

毒消し草(600G)とちからの草(700G)は値段一択なので確定識別可能。

壺の識別チャート（容量で絞る）

押す壺の漢識別は次フロアの階段上で（魔物の壺対策）。店内で水鉄砲・トドは事故のもと。

押せる壺

  • 容量2 → 笑いの壺で確定
  • 容量3以上 → 押して識別

入れる壺の判別フロー

  • 「入れる」でやり過ごし状態に → やりすごしの壺（呪われていても「入れられなかった」で判別可）
  • 容量2（弱化・強化・おはらい・呪い・倉庫・底抜け）→ 不要な識別済み装備を入れる：
    • 消えた→店外で割って落とし穴が出れば底抜け、何もなければ倉庫
    • 呪われた→呪いの壺 ／ ？が消えた・解呪された→おはらいの壺
    • 変化なし→次フロアで強化⇔弱化を確認
  • 容量5（保存・識別・変化・換金・手封じ・倉庫・ただ・割れない）→ 不要品を入れる：
    • 取り出せる→保存 ／ 識別された→識別 ／ 変わった→変化 ／ ギタン化→換金
    • 手封じ状態→手封じ ／ 消えた→倉庫
    • 何も起きない→「ただの壺」と命名して壁に投げる。割れたらただの壺、割れなければ割れない壺
  • 容量3-4 → 同種アイテムが合成されたら合成の壺（値段でも一意に確定可能）

基本価格1000G台の壺は「入れた物を失う or 手封じ」の要注意価格帯。

強化の壺は使用済みの杖を入れるとフロア毎に回数が増えるので判別できる。

武器・盾・お香の識別

武器・盾

  • 装備すれば強化値と呪いが判明。呪い対策に、装備はずしの罠・おはらいの巻物がある時に試すこと。
  • 未識別のまま合成しても識別される（呪われる回数を減らせる）。
  • 値段の下1桁が0以外 → ほぼ呪われている。
  • かつおぶしはかじると識別される。
  • 旅仲間のアスカに装備を投げると識別される（呪い判別にも使える）。

お香

  • 焚けば識別されるが値段が全て同じなので値段識別は不可。漢識別はフロア移動の直前に。

仕組みメモ
  • 斜めに投げたアイテムがボヨヨン壁の上下左右4面すべてで「く」の字反射すると増殖する。
  • 反射後10マス以内に次の壁に当たらないとその場に落ちる（大部屋では不成立になりやすい）。
  • 反射は最大24回まで。投擲物がシレンのマスに戻ると足元でキャッチする。
  • 部屋の四隅に直撃した場合は反射せずその場に落ちる。
  • 遠投状態では壁を貫通してしまうため増殖不可。増殖成功後、壁は数回で壊れる。
